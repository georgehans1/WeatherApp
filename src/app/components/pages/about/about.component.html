About the WeatherApp

A simple WeatherApp built using Angular and the OpenWeatherAPI

inspired by Aisha's figma design and the iPhone's weather app

Home

The purpose of the home page is to show the weather details of the user's current location. The home page consists of 4 different components. A header which has a hamburger menu toggle, a card which contains weather information,a mini card which has been duplicated 4 times to show certain weather information and a toggle button which switches between Celcius unit and Fahrenheit unit. The home component uses a Geolocation API which gets the location of the current device you're using and send this location to the OpenWeatherAPI to then receive information about the weather in your current location.

Search

The Search consists of 4 components. A header, a card,mini cards and a toggle button just like the home page. It also has a search bar that allows you to search any location in the world and get the details of the weather in that particular location.
